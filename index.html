Introduction

About the Project

Hello! My name is Ashwin Kirubakaran and I am extremely excited to share with you my Birthday Circuit Project. This was a final project for the exploratory section of my Electrical and Computer Engineering class, where I had the opportunity to design and create a circuit that would automatically show my birthday. To create this, I used various components, such as 7-segment displays, integrated circuits (ICs), potentiometers, and much more. Building this project took about a week, during which I gained numerous skills.

Step 1

The first step to building this project was to do calculations. I had to first generate the necessary truth table for this design. The truth table that I had designed shows each combination of X, Y, and Z(the inputs), what these inputs are supposed to display on the 7-segment display, and which segments will light up for each number (the outputs).

After creating the truth table, I had to generate all the necessary Boolean equations, and simplify them, using K-Mapping (Karnaugh Mapping). K-Mapping is a technique used to simplify Boolean expressions, which makes it easier to design efficient circuits. A K-Map is a grid representation of a truth table, where each cell corresponds to a unique combination of input conditions, and their resulting output (true or false) in Boolean form.

Step 2

In this part of the project, I had to write a program that takes 3 digital inputs, and uses conditional logic to light up specific segments of a 7-segment display. Each combination of inputs gives a different segment pattern on the display, which shows the different numbers of my birthday. In my program, I had 2 important sections, being the setup section, and the loop section. For the Setup Section, I configured pins 12, 11, and 10 on the Arduino Nano as input pins, and pins 2 to 8 as output pins that control the 7-segment display. Following this, I wrote the loop section, which runs repeatedly, and updates the display segments according to the input states. In this, I made commands that read the current state (HIGH or LOW) of each input pin and stored them in the specific variables (i.e. input0). In this section, I also used if/else if conditions to check each possible combination of the three inputs, corresponding to my birthday. Finally, I tested my code with a simple circuit that included an Arduino Nano and a 7-segment display.

Step 3

In the third part of this project, I integrated a 74193 4-bit up/down counter into the circuit. I connected the counter to a pulser on the trainer board. Each time I toggled the pulser, the counter would display a different number or dash related to my birthday on the 7-segment display. Additionally, using the reset function I implemented, I could reset the counter to the starting number of my birthday, and view the birthday sequence once again.

Step 4 & 5

In the 4th step of my project, I focused on automating the circuit to display my birthday without human intervention. To do this, I had to remove the connections to the pulser, and enable the circuit to operate by itself. I used a potentiometer, as well as a 555 timer, which allowed me to adjust the speed at which my birthday was shown on the display. Additionally, for step 5, I had to enhance the aesthetics of the circuit by ensuring that all wiring was neat and organized, making sure there were no crosses/bridges.
